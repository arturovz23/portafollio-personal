App TODO React

Code:

To make this app I used the following tools: HTML-CSS-JavaScript-React-Git

This app was created using JSX, separating the different sections of the app into components. 'UseState' was used to change application states, 'UseEffect' to save the state into Local Storage.

Does it work?

You can add any task in the text box of the app and add it with enter or by clicking on the '+' icon. The tasks will be stored in a list and you can modify or delete them whenever you want.

When you do a task you can give it a 'check' and it will be removed from the pending list. You can see the completed tasks by clicking the 'show completed' button.
